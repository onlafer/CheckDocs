br
На интеллектуальной панели в надстройках Word появятся кнопки:
 1. Расставить ссылки (связывает  фрагмент текста вашего документа с  материалами справочной системы);
 2. Проверить ссылки (проверяет актуальность уже расставленных в вашем документе гиперссылок на материалы систем "Кодекс"/"Техэксперт" и применяет к ним ряд операций);
 3. Строка поиска (запускает "интеллектуальный поиск");
 4. Кнопка вызова справки;
 5. Список источников (какая из систем "Кодекс/Техэксперт" используется для интеграции);
 6. Кнопка перехода в систему.
br
div
br
1.Интеграция может не подклключиться, если были запущены приложения во время установки. 
2. Интеграция с CAD-программами возможна при наличии прав администратора
3. Некорректная распаковка архива (все файлы должны распаковаться в папку kAssist) 
4. Ошибка при скачивании архива, архив пустой или неполный. Нужно скачать заново (размер архива более 30 мб.). 
5. Чтобы отключить kAssist, нужно заново запустить файл setup.exe, перейти во вкладку "Приложения для интеграции" и передвинуть ползунок в положение "выключить". То же самое нужно сделать и при переустановке kAssist. 
7. Если надстройка интеграции появилась только в одной программе, то нужно перезапустить ПК.

Если установка прошла успешно, вы увидите на своем экране:(СКРИН)

1 ШАГ. Расставить ссылки.
Нажмите на кнопку «Расставить  ссылки» на интеллектуальной панели. В появившемся диалоговом окне отобразятся обработанные фрагменты  текста, подходящие для установки гиперссылок.
В результате полуавтоматической расстановки ссылки будут отличаться по цвету в зависимости от статуса документа:
синий цвет — для действующих документов;
тёмно-красный — для недействующих документов;
оранжевый — для документов с особым статусом. 					     
По итогам анализа текста все гиперссылки будут разделены на 4 группы:

Автоматически расставленные гиперссылки (данные фрагменты текста имеют только один подходящий документ) (ДОБАВИТЬ КАРТИНКУ)

Фрагменты, имеющие несколько подходящих документов (нужный документ можно выбрать вручную из списка) (ДОБАВИТЬ КАРТИНКУ)

Фрагменты, не имеющие подходящих документов (если выделить документ в тексте курсором, то можно запустить интеллектуальный поиск вручную)
Так как иногда kAssist не распознает атрибуты документа, сформулированные некорректно. (ДОБАВИТЬ КАРТИНКУ)

Фрагменты, имеющие сторонние ссылки (kAssist предложит удалить ссылки на сторонние ресурсы или заменить на гиперссылки систем "Кодекс"/"Техэксперт")

2 ШАГ. После обработки всех ссылок нужно запустить проверку ссылок
br
Нажмите на кнопку "Проверить ссылки". Появится окно, содержащее полную информацию о гиперссылках.
По итогам анализа ссылки будут разделены на 4 группы в окне проверки (ДОБАВИТЬ КАРТИНКУ)
br
Ссылки разделены на 4 группы:
br
div
br
Ссылки на действующие документы: эти ссылки не требуют дополнительной обработки, кроме правовых документов.kAssist по умолчанию ставит ссылки на ту редакцию Кодекса (или ФЗ), которая действует на данный момент. Чтобы проверить актуальность редакции нужно перейти в текст документа и возпользоваться сервисом "документ во времени". СКРИНШОТ
br
div
br
Ссылки на недействующие документы. Для каждого документа из списка реализованы функции: Переход к тексту документа; Указан период действия документа; Указана действующая редакции документа с возможностью замены ссылки; Аналитическая работа с документами  СКРИНШОТ
br
div
br
Ссылки с особым применением. Для каждого документа из списка реализованы функции: Переход к тексту документа (ознакомление с особенностями применения); Указан период действия документа; Указаны связанные документы; Переход в службу поддержки пользователя. СКРИНШОТ
br
div
br
Ссылки с неактуальной информацией. В диалоговом окне необходимо обновить ссылку, пока она не изменит цвет (синий - действующий документ, красный - отмененнный документ, оранжевый - документ с особым статусом). Дальнейшая работа см. пункты 1-3 СКРИНШОТ
br
div
br
Дополнительная информация
1. Опции для самостоятельной работы со ссылками:
а) просмотр списка установленных гиперссылок;
б) открытие документов в системе или удаление гиперссылки;
в) выделение нужных гиперссылок цветом;
г) ручной поиск нужного документа в системе, на который будет направлена ссылка. СКРИНШОТ

В случае, когда во фрагментах текста есть опечатки, можно просмотреть список данных фрагментов и выбрать дальнейшее действие: воспользоваться поиском по системе «Кодекс»/«Техэксперт» для уточнения информации и самостоятельного создания
гиперссылки или оставить фрагмент текста без внимания.

Пример проверки редакций 1. Перейти в актуальную редакцию Кодекса. 2. Открыть меню. 3. Запустить сервис "Документ во времени". 4. Выбрать нужную дату (когда была написана ВКР или другой текст). 5. Система откроет ту редакцию, которая действовала в эту дату в прошлом. 6. Сравнить эту редакцию документа и актуальную (вкладка "Редакции", подробнее - http://project5381457.tilda.ws/page27082427.html). 7. Если времени нет, то скопировать ссылку на старую редакцию и поставить ее в текст вручную. Проверить позже. (пункт "установка ссылок вручную")
br
